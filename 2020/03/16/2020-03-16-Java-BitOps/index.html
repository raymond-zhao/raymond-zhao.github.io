1. 位操作符

  • & 与运算符：全1为1，反之为0.
  • | 或运算符：全0为0，反之为1；只要一个位为1，结果为1.
  • ^ 异或运算符：两个位相同则为 0，不同则为 1
  • ~ 取反运算符：0 则变为 1，1 则变为 0
  • << 左移运算符：向左进行移位操作，高位丢弃，低位补 0
  • >> 右移运算符：向右进行移位操作，对无符号数，高位补 0，对于有符号数，高位补符号位
